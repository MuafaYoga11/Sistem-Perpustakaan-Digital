# 📚 Sistem Perpustakaan Digital

[![Contributions Welcome](https://img.shields.io/badge/contributions-welcome-brightgreen.svg?style=flat)](💡)
[![Laragon Supported](https://img.shields.io/badge/Environment-Laragon-blue?style=flat&logo=laragon)](https://laragon.org)

Sistem Perpustakaan Digital adalah aplikasi berbasis web yang dirancang untuk mempermudah pengelolaan data buku, anggota, serta transaksi peminjaman dan pengembalian buku secara efisien dan terstruktur.

---

## ✨ Fitur Utama

Aplikasi ini dilengkapi dengan berbagai fitur esensial untuk manajemen perpustakaan:
*   🔐 **Autentikasi Multi-User**: Login untuk Admin dan Anggota.
*   📖 **Manajemen Buku (CRUD)**: Tambah, lihat, ubah, dan hapus data buku beserta kategorinya.
*   👥 **Manajemen Anggota**: Pengelolaan data pengguna/siswa yang terdaftar.
*   🔄 **Transaksi Peminjaman**: Pencatatan otomatis tanggal pinjam, batas waktu, dan status pengembalian.
*   💰 **Sistem Denda Otomatis**: Menghitung denda keterlambatan pengembalian buku.

---
